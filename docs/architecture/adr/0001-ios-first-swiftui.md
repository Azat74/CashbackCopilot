# ADR-0001: iOS-first app built with SwiftUI

## Status

Accepted

## Context

Нужен нативный iOS MVP с камерой, локальным state management и быстрым UX перед оплатой.

## Decision

Строим iOS-first приложение на SwiftUI.

## Consequences

- плюс: нативный UX и прямой доступ к Apple frameworks
- минус: выше входной порог и зависимость от Xcode tooling
