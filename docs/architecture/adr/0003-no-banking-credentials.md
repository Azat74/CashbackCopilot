# ADR-0003: No banking credentials or direct bank integrations in v1

## Status

Accepted

## Context

Доступ к банковым аккаунтам не нужен для проверки core value продукта и сильно расширяет security scope.

## Decision

В MVP не собираем и не обрабатываем банковские креды или issuer tokens.

## Consequences

- плюс: ниже юридические и security риски
- минус: больше ручного ввода на старте
