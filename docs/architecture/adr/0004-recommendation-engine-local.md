# ADR-0004: Recommendation engine runs locally on device

## Status

Accepted

## Context

Расчет должен быть быстрым и доступным прямо в момент оплаты, без сетевой зависимости.

## Decision

Recommendation engine работает полностью на устройстве.

## Consequences

- плюс: low latency и privacy by design
- минус: меньше централизованной observability
