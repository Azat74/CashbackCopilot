# Glossary

- **Advisory recommendation** — рекомендованный способ оплаты, а не гарантия банкового результата.
- **Cashback rule** — пользовательское правило, описывающее когда способ оплаты дает выгоду.
- **Confidence** — степень уверенности системы в корректности рекомендации.
- **Payment channel** — способ проведения оплаты: карта, QR, SBP.
- **Purchase context** — набор входных данных для расчета: сумма, категория, merchant, канал оплаты.
- **Reward cap** — лимит кешбека в денежном выражении.
- **Spend cap** — лимит оборота, который еще участвует в расчете кешбека.
- **Actual cashback** — фактически начисленный кешбек, подтвержденный пользователем.
