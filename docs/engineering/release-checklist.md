# Release Checklist

## Product

- критические user flows проверены вручную
- disclaimer и uncertainty language соответствуют реальному поведению
- empty states и failure states не выглядят сломанными

## Engineering

- build passes
- tests pass
- archive build succeeds

## Security / Privacy

- нет секретов в repo
- нет банковских credentials в app scope
- permissions и copy проверены
- нет случайной внешней телеметрии
