# Definition of Done

Фича считается готовой, если:

- acceptance criteria выполнены
- happy path работает
- основные edge cases покрыты тестами или явно задокументированы
- не нарушены business logic guardrails
- проект собирается
- тесты проходят
- документация обновлена, если менялась доменная семантика
- security/privacy assumptions не нарушены
